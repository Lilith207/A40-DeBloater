### This is a debloater for the samsung A40
This module gets "rid" of the following apps / packages:
- Facebook and its services
- Galaxy stores
- AR apps like emoji and AR zone
- Onedrive
- Youtube
- Calendar app (the default one)
- Digital wellbeing app

  These are just some examples, for the full list look into bloat_packagelist.txt
  Keep in mind this swings a bat at the kneecaps of the "intended" way to use a samsung phone. 
  Most default stuff can break and your phone will not have the same functions.
  It should still be perfectly usable if you use alternative apps though (see fdroid).
